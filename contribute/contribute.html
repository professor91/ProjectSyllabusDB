Code webpage for a course

Get the latest syllabus pdf (2019-2020) of a programme from the list of given courses and code webpage for it. Use the Bsc CS Hons source code for the reference.

For this you need basic knowledge of HTML & CSS. If you don't know these use this playlist to learn. I have handpicked all the videos in this playlist. All that has been used in making of the project will be available in the playlist.

In the project website of each programme (BSc/BA/BComm) is divided into two parts:

  • Main page- which consists of Course Timeline Distribution and list of all the courses involved in the programme
  • Course page- A seperate webpage dedicated to each course consists of it's detailed syllabus and other resources

Main page for some courses has already been coded, to track the progrss of the project keep an eye on this Google Sheet. Choose what part of the website you wanna code only after looking at this google sheet.
